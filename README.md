# qFioofa-tmux

Personal tmux config.

# Install

Install tmux

```bash
sudo apt install tmux
```

Plugin manager
```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Clone repo

```bash
git clone https://github.com/qFioofa/qFioofa-tmux
```

Deploy config

```bash
cd ./qFioofa-tmux
bash ./scripts/deploy.sh -r
```
